➕ Buat Automation Rule

🪄 Wizard Rule Operasional

Pilih template bahasa operasional atau tulis instruksi kerja Anda sendiri. Wizard akan mengisi visual builder secara otomatis.

Wizard akan membaca kata kerja seperti impor, permission, assign, queue campaign, delivery, scan recovery, requeue, cooldown, retry, akun online, dan batas per siklus.

🧱 Visual Rule Builder

Susun rule tanpa menulis JSON. Semua pilihan di bawah akan otomatis diubah menjadi Condition, Action, dan Scope yang valid.

Buka mode lanjutan JSON

🧩 Edit Rule

🪄 Wizard Rule Operasional

Gunakan bahasa operasional untuk mengubah rule yang sedang dibuka. Stage rule tetap mengikuti rule ini dan wizard akan menyesuaikan field-fieldnya.

Untuk edit rule, wizard mengikuti stage rule yang sedang dibuka lalu mengisi field yang relevan secara otomatis.

🧱 Visual Rule Builder

Ubah rule dengan dropdown, checkbox, dan angka. JSON tetap tersedia hanya untuk kebutuhan lanjutan.
